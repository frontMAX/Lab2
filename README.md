# User Based Content 
I detta projekt kommer vi skapa en `User Based Content` med hjälp av `MongoDB`.

<br>

## Beskrivning för att komma igång
Skriv i terminalen `npm i`
Sedan `npm start`.
Öppna därefter upp din browser och skriv in http://localhost:3000
